# ① 課題番号-プロダクト名

セカイドア

## ② 課題内容（どんな作品か）

- 外国人材採用版の価格.com的なサイトです。
- 良質な人材を求めている企業、日本で働き技術や知識を身につけたい外国人、優れた人材を紹介したい監理組合、登録支援機関。みんながハッピーになれる三方良しのシステムです。
